# NirvanaRedirect

## 介绍
涅槃科技官方重定向中转开源，让您的跳转不在一样，非同一般。

## 使用说明
修改[NPyyds.js](js%2FNPyyds.js)中的<br>---------------------------------<br>
title 标题内容<br>
imgSrc 图片链接<br>
cssHref 背景地址<br>
url 跳转链接【可选】<br>
----------涅---槃--科--技--------------

## 使用介绍
1. 在HTML中自动嵌入加载动画<br>
2. 加载动画后跳转至其它链接达到中转
